# MMA Fight Analysis & Fighter Skill Tree

HAI GUYS :3 this is my project to get better at machine learning for event recognition pose estimation. This started as an idead I had while training muay thai and bjj. What if I could make a way to make a skill tree for ufc fighters like a pokemon rolledex hmmm????? welp sooo here I am. Also big shoutout to JABBR as they're technology and reserach papers are a big reason why I tought of this.

Multi-stage pipeline: detect fighters, estimate pose, classify stance, recognize strikes, build a per-fighter skill tree. See specupdates for full design, stateupdates for current status, decisionsupdates for the decision log.

**Dataset:** Mendeley `c456bnk8bm` (V2), CC BY-NC-SA 4.0 — https://data.mendeley.com/datasets/c456bnk8bm/2; download separately.
**Status:** Phase 1 (detection + pose) trained on YOLO11n-pose.
